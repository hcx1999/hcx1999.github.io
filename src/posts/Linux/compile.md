## 编译指令

### python

```bash
python3 file.py
python3 -m pdb file.py
```

[pdb的用法](https://zhuanlan.zhihu.com/p/37294138)

### C++

```bash
g++ file.cpp -o file
./file

g++ -g file.cpp -o file
gdb file
```

[g++编译选项](https://blog.csdn.net/m0_47406832/article/details/133798293)，[gdb的用法](https://blog.csdn.net/weixin_45031801/article/details/134399664)
