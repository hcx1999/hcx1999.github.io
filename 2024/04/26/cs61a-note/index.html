今明两天，集中突破！

a = s 是对s进行浅拷贝，a与s地址相同，a is s返回True，除非s或a中的一个被再次赋值，否则a将一直随s变化

b = s[:] 是对s进行深拷贝，b == s返回True，b is s返回False，对于一维list来讲，已经完成了物理拷贝

而对于多维list来说，深拷贝也可能偷懒把list的地址换了但是里面元素的地址不更换（元素也是list类型），如果有必要可以使用deepcopy()函数:c = deepcopy(s)


list的元素甚至可以是None！type(None) == NoneType

python的iterator用完都不归位的么，把他放到自变量里面执行一次之后值就变了，跟那些容器一样。
